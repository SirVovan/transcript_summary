# project.md — Konspekt Project

## Что это

Двойной проект: инструмент для обработки транскриптов видеолекций в структурированные учебные материалы — и одновременно живой скилл, который дорабатывается и улучшается по ходу работы.

Получает на вход текстовый транскрипт (и опционально презентацию) — выдаёт три готовых файла для самостоятельного изучения. Параллельно накапливает договорённости, правила и эталоны в папке `references/`, чтобы каждая следующая сессия работала лучше предыдущей.

## Для кого

Для студентов и учеников онлайн-курсов, которые хотят иметь удобные конспекты видеоуроков — без необходимости смотреть видео повторно.
Прежде всего — для обработки уроков курса «Вайбкодинг» (спикер Дмитрий Ледовских).

## Проблема

Видеолекции неудобно перечитывать и возвращаться к ним за конкретной мыслью. Транскрипты содержат шум и паразитные элементы. Вручную структурировать урок — долго. Этот проект автоматизирует весь цикл обработки.

## Основные функции

**Производство конспектов:**
1. **Карта смыслов (XLSX + HTML)** — таблица сегментов урока с таймингами, типами блоков, сутью, ключевыми мыслями и демонстрациями. Промты из презентации встраиваются прямо в текст.
2. **Конспект (DOCX)** — подробный текстовый конспект от первого лица спикера, без потерь логики и смысла. Формат: Word-документ, готовый к чтению и печати.
3. **Виджет (HTML)** — интерактивная HTML-презентация конспекта. Один экран = один смысловой блок, навигация по вкладкам, правая колонка с визуальными компонентами (статы, схемы, инсайты, промты с кнопкой копирования).

**Развитие скилла (`references/`):**
- `kartograf.md` — правила и этапы создания карты смыслов (ядро + профили по типу контента)
- `kartograf_profile_*.md` — профили для разных типов контента: лекции, кастдев, созвоны, конференции, база
- `konspektolog.md` + `konstitutsiya.md` — правила написания конспекта и голоса спикера
- `widget.md` + `widget_skeleton.html` — правила и каркас виджета
- `karta_example.html` — эталонный HTML-пример карты смыслов
- `xlsx_template.py` — скрипт генерации XLSX

Правила дополняются после каждой сессии: если в ходе работы выработана новая договорённость — она вносится в соответствующий файл.

## Страницы / экраны

- `transcripts/` — входящие транскрипты (.txt, .docx) и сопроводительные презентации (.pptx, .md)
- `transcripts/[урок]_карта_смыслов.xlsx` — карта в Excel
- `transcripts/[урок]_карта_смыслов.html` — карта в HTML
- `transcripts/Конспект — [урок].docx` — текстовый конспект
- `transcripts/Виджет — [урок].html` — интерактивный виджет

## Чего НЕ будет (пока)

- Автоматической транскрибации аудио/видео — транскрипт подаётся вручную
- Веб-интерфейса или деплоя — всё работает локально через Claude Code
- Поддержки других языков, кроме русского
- Автоматической публикации или отправки файлов
