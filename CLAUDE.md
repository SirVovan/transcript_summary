# Konspekt Project

> При начале работы прочитай `D:\Users\Вова\Desktop\Work\VibeCoding\vibecoding vault\projects\konspekt-project.md` — там статус и следующий шаг.

Этот проект обрабатывает транскрипты видеолекций в учебные материалы.

## Как работать

Когда пользователь даёт транскрипт или просит создать карту смыслов / конспект / виджет —
использовать скилл `/konspekt`. Он содержит полный пайплайн и все необходимые инструкции.

Все рабочие файлы скилла находятся в `.claude/skills/konspekt/`.
Выходные файлы сохраняются в `transcripts/`.
