ПРОФИЛЬ: CONFERENCE
━━━━━━━━
Для: конференции, панельные дискуссии, онлайн-практикумы,
митапы и мероприятия с несколькими спикерами.

Принцип сегментации:
Делить по смене спикера и крупным смысловым темам.
Не дробить внутри одного выступления на мелкие подтемы —
они описываются в колонке «Суть».

Минимальная длина сегмента: 5–7 минут.
Исключение — только орг. блоки с самостоятельным содержанием:
розыгрыш, технический перерыв, заполнение анкеты, голосование.
НЕ являются исключением:
- анонс следующего спикера (включать в его выступление)
- слова благодарности предыдущему спикеру (включать в его сегмент)
- переходные реплики длиной 1–3 минуты между выступлениями
  (включать в предыдущий или следующий сегмент по смыслу)

Ориентиры для границ сегмента:
- Смена спикера
- Принципиально новая крупная тема внутри выступления одного спикера
- Переход от содержательного блока к организационному (и обратно)
- Начало Q&A или панельной дискуссии

Не создавать отдельный сегмент для:
- Каждого подпункта, шага или роли внутри одного блока
- Коротких переходных реплик (1–3 минуты)
- Анонсов и представлений спикеров без самостоятельного содержания
- Мелких смысловых оттенков внутри одной темы

Дополнительные колонки:
Колонку «Трек / секция» добавлять только если в транскрипте
есть явные маркеры именованных секций — заголовки, объявления
типа «открываем блок X» — И пользователь явно подтвердил
её необходимость. Без этих двух условий — не добавлять
и не спрашивать.

Вопрос для Шага 3:
«Уточни имена ведущего и основного спикера,
если они не названы в транскрипте явно.»

Допустимые значения «Тип блока»:
- Открытие / приветствие  — начало мероприятия или дня
- Концепция               — спикер вводит идею, объясняет «почему» и «зачем»
- Методология             — спикер даёт пошаговую систему, алгоритм, фреймворк
- Кейс / пример           — разбор конкретного случая, история с результатом
- Демонстрация            — работа с инструментом в прямом эфире / показ экрана
- Панельная дискуссия     — несколько спикеров обсуждают тему
- Вопросы / ответы        — Q&A с аудиторией
- Маркетинговый блок      — презентация продукта, оффер, тарифы, призыв к покупке
- Орг. момент             — технические и организационные вставки (розыгрыш,
                            перерыв, анкета); не использовать для переходных реплик
- Закрытие                — завершение дня или мероприятия

Тип «Выступление» не использовать — он слишком широкий.
Если содержание сегмента не укладывается ни в один тип —
выбрать ближайший и добавить уточнение через «/»
(например: «Концепция / мотивация», «Методология / демонстрация»).
