ПРОФИЛЬ: LECTURE
━━━━━━━━
Для: лекции, обучающие видео, вебинары, мастер-классы, спринты.

Принцип сегментации:
Делить по смысловым темам и переходам между ними.
Ориентир — смена темы, введение нового понятия, переход к практике.

Допустимые значения «Тип блока»:
- Введение
- Концепция
- Методология
- Инструмент / практика
- Пример / кейс
- Демонстрация
- Задание / ДЗ
- Вопросы / ответы
- Орг. момент
- Итоги / резюме
