ПРОФИЛЬ: MEETING
━━━━━━━━
Для: рабочие созвоны, планёрки, командные встречи.

Принцип сегментации:
Делить по повестке и переходам между темами.
Ориентир — смена вопроса повестки, принятое решение, появление нового участника.

Дополнительные колонки (добавляются перед «Суть»):
| Решение / action |

Допустимые значения «Тип блока»:
- Открытие / повестка
- Обсуждение
- Проблема
- Решение
- Голосование / согласование
- Action item
- Информирование
- Орг. момент
- Закрытие
